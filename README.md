# Tidal Flow

Tidal Flow is a Java based fluent assertion library.

## Downloads

You can download the dependency from maven

``` maven
<dependency>
   <groupId>io.github.tidal-code</groupId>
   <artifactId>flow</artifactId>
   <version>1.0.0</version>
</dependency>
```
